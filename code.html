Введение начнем с теории.

Код мы будем писать на Html с дополнением в css, начнем по порядку.
Возьмем из вики: HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере. Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS или открывают с локального диска, далее интерпретируют код в интерфейс, который будет отображаться на экране монитора.

Если быть кратким, html это файл с кодом задающий строительные блоки. Весть текст хранится в html файле. Html это фундамент статичного сайта.


Опять же определение из гугла:CSS (англ. Cascading Style Sheets «каскадные таблицы стилей») — формальный язык декодирования и описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (чаще всего HTML или XHTML).

И сново кратко. Css это файл оформления, в нем хранится код ставищий задний фон, цвет текста, табы и всё оформление. Без него бы страница выглядела так:

Cамое начало

Сразу хочу сказать, что если у вас есть уже опыт написания и заготовку для сайта вы можете сделать сами, то вы можете пролистнуть эту часть


Сейчас перед вами считай чистый лист, вам же нужно напротив, написать на нем сайт, это будет не настолько сложно как вы можете подумать начнем опять же по порядку.

Пошагово:

1. Мы покажем браузеру что наш сайт написан на HTML, для этого нам надо ввести:
<!doctype HTML>

2. Теперь когда браузер знает как читать наш сайт нам надо начать сделать его самого. Для это пишем теги html:
<html> и </html>

почти каждый тег имеет свою оканчивающию часть, мы так показываем от куда до куда должен применятся тег.

3. Нам нужно открыть головной тег и закрыть, он выглядит так:
<head></head>

этот тег должен быть между html тегами, тег head задает место для головных тегов: название сайта, показание где файл стиля и подобное.

4. Создаем между тегов head:
<title>название вашего сайта</title>

эти два тега задают название страницы, как она будет подписа по переходе по ссылке или же вверху браузера, между этими тегами напишите желаемое название.

5. Теперь нам надо задать файл css, пока что это вовсе отдельный файл, для связи их мы используем команду:
<link rel="stylesheet" href="название вашего css файла.css">

6. Мы задали невидимую чусть нашего сайта, сейчас же мы должны начать писать что то на самой странице, для этого пишем два тега между Html тегами:
<body></body
Между этими тегами мы и будем писать содержание.

К этому шагу у нас должно быть что то наподобии:

7. Если у вас такой же код то продолжаем, далее мы пишем между тегов body само содержание, о возможных командах вы можете найти в mimo обучение, но я приведу минимальный довольно легкий сайт для примера, и так между тегами body мы пишем:
<h1 id="Заглавление">Ваше заглавление<h1>
После него пишем тег:
<br>
Тегом h мы создаем обозначение текста и его размера от 1 до 6, далее у нас идет дополнение команды командой id="" это как надпись на коробке, поможет нам в будущем для указание именно на эту переменую. А тег br стоит что бы этот текст не слипся с следущим

8. Далее ниже мы пишем тег:
<p id="Текст">Любой текст, оснавной мосив информации<p>
Это еще один тег для показа текста, в нём напишите любой нужный текст, что бы текст шёл на другую строчку можно использовать тег:
<br>
Он преносит текст/фото на следущию строчку

9. Теперь я вам покажу как вставлять картинку, вы должны найти желаемую и скопировать её url адрес, далее написать:
<br> для отделенеие от прошлого текста
<img src="url ссылка на изображение" id="Картинка">

План написания css:

1. Тут я разделю на конкретные теги которые мы будем оформлять, первым будет общие оформление, но для начала сверьтесь с моим файлом, html файл у вас должен выглядить примерно так же, а id индентично:

2. После проверки перейдем к написанию, заходя в css файл у вас должно быть пусто, начнем мы с ноформление всего сайта, а именно тега body, сделаем задний фон, выглядить это будет так:
body{background-rgb(55, 88, 88); background-repeat: no-repeat; background-attachment: fixed; background-size: cover; color:aliceblue;}
первый тег ставит задний фон, в виде цвета, rgb вы можете изменить под свои хотелки, 2 тег фиксирует фотку что бы она не листалась вместе с страницей, а 3 тег растягивает её коректно, 4 тег color задает цвет текста

Следущим шагом мы украсим "Заглавление", это будет выглядить так:
#Заглавление{text-align: center;}
#Заглавление мы пишем для показа на что применять это оформление text-align: center задаёт расположение надписии, в нашем случие выравнивание по центру

3.Следущие что мы будем "пытать" с помощью css будет text, выглядить это будет так:
#text{ background-color: #000000; border-radius:0.375rem; padding: 1rem; }
и опять разбор, background-color задает цвет за текстом. border-radius задает округление. padding это отступление. Ваш css код должен выглядить так:
